# Tribute Page – Marie Curie

This is a simple tribute webpage dedicated to Marie Curie, a pioneering physicist and chemist.

## Features

* Basic HTML and CSS layout
* Responsive design for smaller screens
* Alternating text and image sections
* Styled header and footer
* Clean typography using Google Fonts

## Content

The page includes:

* A short introduction about Marie Curie
* Key achievements and life events
* Images alongside descriptive text

## Technologies Used

* HTML
* CSS

## Notes

* Images are stored locally in an `images` folder
* No JavaScript is used in this project
