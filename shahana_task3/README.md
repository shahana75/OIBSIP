# Elixer To-Do Lister

A simple web-based to-do list application to manage daily tasks.

## Features

* Add new tasks
* Mark tasks as completed using a checkbox
* Move tasks between incomplete and completed sections
* Delete tasks permanently
* Add tasks using button or Enter key

## Structure

* Input field to enter task description
* "Add" button to add tasks
* Two sections:

  * Incomplete Tasks
  * Completed Tasks

## Technologies Used

* HTML
* CSS
* JavaScript (DOM manipulation)

## Notes

* Tasks are not stored permanently (no database or local storage)
* Simple UI with basic styling
* Uses emoji for delete button
