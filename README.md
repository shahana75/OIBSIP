## Web Development Internship Tasks

This repository contains the projects completed as part of my **Web Development Internship at Oasis Infobyte**. Each project focuses on strengthening core frontend and basic backend concepts.

---

## Projects Included


### Calculator Web App

A simple calculator that performs basic arithmetic operations.

* Supports +, −, ×, ÷
* Keyboard input enabled
* Interactive UI

---

### Tribute Page

A responsive tribute page dedicated to a notable personality.

* Structured content layout
* Image and text sections
* Clean and aesthetic design

---

### To-Do List

A task management app to organize daily activities.

* Add and delete tasks
* Dynamic updates using JavaScript
* Simple and user-friendly interface

---

### Login Authentication

Basic login system with validation.

* User input handling
* Form validation
* Authentication logic (frontend/backend based)
