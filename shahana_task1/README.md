 Calculator Web App

A simple and interactive calculator built using HTML, CSS, and JavaScript. It performs basic arithmetic operations with a clean UI.

-----

 Features
Basic operations (+, −, ×, ÷)
Responsive button layout using CSS Grid
Keyboard support
Clear & delete functionality
Smooth UI with hover and click effects

-----

 Tech Stack
HTML
CSS
JavaScript

-----

 How to Run
Download the project
Open index.html in browser
